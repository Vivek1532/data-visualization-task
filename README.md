## Task 2: Data Visualization and Storytelling

This project focuses on creating an interactive Power BI dashboard using a Superstore sales dataset.

## Dashboard Features

- KPI cards showing total sales and profit
- Sales by category bar chart
- Profit by region column chart
- Monthly sales trend line chart
- Category breakdown pie chart
- Interactive slicers for category and region filtering

## Insights

- Technology category generates the highest sales
- West region has the highest profit
- Sales fluctuate across months with peaks in Q3
- Category distribution is balanced across segments

## Tools Used

- Power BI Desktop
- CSV dataset
- Data storytelling principles
